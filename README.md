ploutos-deployment
==================

deploy ploutos


###Configuration
---
* rel/vars.config

###Compiling
---
```bash
make
```

###Generate Node
---
```bash
make rel
```

###Install
---
```bash
make install
```

###Uninstall
---
```bash
make uninstall
```

###Usage
---
```bash
ploutos {start | stop | restart | reboot | ping | console | attach}
```
